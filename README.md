# AI-Scammer-Detection

# Pain Point: ภัยไซเบอร์ในปัจจุบันมักมาในรูปแบบของ "Social Engineering" เช่น การปลอมตัวเป็นเจ้าหน้าที่ธนาคารส่งข้อความมาข่มขู่หรือให้รางวัล เพื่อให้เหยื่อตกใจแล้วโอนเงิน โดยเหยื่อไม่มีเครื่องมือช่วยตรวจสอบข้อเท็จจริง

# ข้อจำกัดของวิธีปัจจุบัน: ปัจจุบันเมื่อผู้ใช้สงสัย วิธีแก้ปัญหาคือการโทรหา Call Center ซึ่งใช้เวลาและทำได้ยากในสถานการณ์ 

# เหตุผลที่ต้องใช้ Machine Learning (AI): โครงสร้างของภาษาไทยมีความซับซ้อน การตั้งเงื่อนไขโปรแกรมแบบดั้งเดิม (If-Else) ไม่สามารถครอบคลุมทุกประโยคที่มิจฉาชีพแต่งขึ้นมาใหม่ได้ จึงจำเป็นต้องใช้ Machine Learning (NLP) เพื่อทำความเข้าใจเจตนา ของข้อความ และใช้ RAG เพื่อดึงนโยบายจริงของธนาคารมาเปรียบเทียบหาความขัดแย้งร่วมกั
