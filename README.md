# AI-Scammer-Detection

### ในปัจจุบัน ภัยคุกคามทางไซเบอร์ในหลายๆวิธีเพิ่มขึ้นเป็นอย่างมาก โดยเฉพาะการที่แสกมเมอร์ได้แอบอ้างเป็นหน่วยงานที่น่าเชื่อถือ แล้วทำให้มีผู้ใช้งานบนโลกอินเตอร์เน็ตมากมายที่ตกเป็นเหยื่อให้กับแสกมเมอร์ในการถูกหลอกลวงผ่านทาง SMS ให้โอนข้อมูลส่วนตัวหรือทรัพย์สินไป ซึ่งในปัจจุบันการมีอยู่ของโมเดล AI ไม่ว่าจะเป็นโมเดล Natural Language Processing(NLP) หรือ Large Language Model(LLM) ที่มีการพัฒนาให้มีประสิทธิภาพสูงขึ้นเป็นอย่างมาก และด้วยความที่ว่าแอพหรือโปรแกรมตรวจจับ SMS จากแสกมเมอร์ที่เป็นภาษาไทยพร้อมให้โมเดล LLM วิเคราะห์ข้อความยังมีไม่มากนัก ด้วยเหตุนี้ผมจึงคิดที่จะทำระบบ AI ที่จะเริ่มต้นด้วยการให้ NLP ตรวจจับ sms จาก แสมเมอร์จากนั้น จะส่งให้ LLM วิเคราห์และให้เหตุผล โดยจะมีตัวช่วยอย่าง Retrieval-Augmented Generation (RAG) ในการดึงนโยบายต่างๆของแต่ละบริษัทหรือองค์กร ว่ามีนโยบายในการจัดส่ง SMS ไปหา User ด้วยรูปแบบแบบไหนบ้าง
